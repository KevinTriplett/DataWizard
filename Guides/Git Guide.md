# Git Guide

A complete reference for using git with Obsidian vaults and DataWizard projects. Covers setup, daily workflow, collaboration, safety, and recovery.

**Audience:** Anyone working with a git-synced Obsidian vault — whether you're backing up a personal vault, collaborating on a shared project, or pulling DataWizard Seed updates.

**How to use this guide:** Read the sections relevant to your situation. If you're setting up for the first time, work through Sections 1-3, then Section 5.5 (DW Save) to set up your save hotkey. If you're already set up and need to collaborate, jump to Sections 5-6. If something went wrong, go to Section 7 or 9.

**For Claude instances:** The git-onboarding skill (`Seed/Skills/git-onboarding/SKILL.md`) provides an interactive walkthrough that uses this guide as its reference. Load the skill when onboarding a new collaborator.
