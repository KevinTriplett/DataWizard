# 1.0 Prerequisites and Install

Before setting up git, confirm these are in place.

## What You Need

- **A GitHub account** -- sign up at https://github.com if you don't have one
- **Git installed** -- see platform-specific instructions below
- **A terminal** -- Terminal on Mac, PowerShell or Git Bash on Windows

## macOS

Install Xcode Command Line Tools (includes git):

```bash
xcode-select --install
```

Verify:

```bash
git --version
```

If you prefer Homebrew: `brew install git` works too. Either way, note the full path for later (needed for the Obsidian Git plugin):

```bash
which git
```

(Usually `/usr/bin/git` or `/opt/homebrew/bin/git`)

## Windows

Download and install Git for Windows from https://gitforwindows.org. Use the default settings during installation.

After install, open PowerShell and verify:

```powershell
git --version
```

Git for Windows also installs Git Bash, which provides a Unix-like terminal. Either PowerShell or Git Bash works for the commands in this guide.

## Set Your Identity

Git tags every commit with your name and email. Set these once:

```bash
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"
```

Verify:

```bash
git config --global user.name
git config --global user.email
```

## GitHub CLI (Optional)

The GitHub CLI simplifies authentication and repo creation. Not required -- you can do everything through the GitHub web interface and standard git commands.

- **Mac:** `brew install gh`
- **Windows:** `winget install --id GitHub.cli` or download from https://cli.github.com

Authenticate: `gh auth login` (follow the prompts).
